# Drip Store

Projeto de e-commerce em React com Vite e Tailwind, estruturado em páginas e componentes reutilizáveis.

## Páginas

- Home: `/`
- Listagem de produtos: `/produtos`
- Visualização do produto: `/produto/:id`

## Tecnologias

- React
- Vite
- Tailwind CSS

## Scripts

```bash
npm install
npm run dev
npm run lint
npm run build
npm run preview
```
